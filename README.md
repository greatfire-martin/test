## 自由浏览都支持哪些设备？
自由浏览目前支持Android 4.0以上，使用ARM处理器的手机、平板电脑等设备，暂不支持使用Intel x86处理器的设备。

## 基本操作说明
新版自由浏览基于谷歌Chrome的开源项目Chromium，基本使用说明请参考<a href="https://support.google.com/chrome/topic/3422804">谷歌Chrome帮助中心</a>。
